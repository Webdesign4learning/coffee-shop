# Coffee Shop Website

This repository contains three different Coffee Shop websites: index, index1, and index2. Each website is built using Bootstrap, CSS, HTML5, jQuery, and Popper.js. They feature three different appealing menu items, a slider showcasing different coffee items, contact us info, a reservation form, featured dishes, client testimonials, and a gallery. <br>

## Installation
To get started with Restaurant Explorer, follow these steps:

1. Clone the repository: `git clone https://github.com/maryamalsadat-tabatabaei/Coffee-Shop-Website.git`
2. Navigate to the project directory: `cd Coffee-Shop-Website`
3. Install dependencies: `npm install`
4. Start the application: Open the desired website in a web browser by double-clicking the corresponding HTML file (index.html, index1.html, or index2.html).

## features

**Key Technologies**
- Bootstrap: Bootstrap is a popular CSS framework that provides a set of pre-built components and responsive design features, making it easy to create visually appealing and mobile-friendly websites.
- CSS: Cascading Style Sheets (CSS) are used to customize the appearance of the websites, ensuring a consistent and visually appealing design.
- HTML5: HTML5 is the markup language used to structure the content of the websites and define their layouts.
- jQuery: jQuery is a JavaScript library used for DOM manipulation, event handling, and animation, enhancing the interactivity and functionality of websites.
- Popper.js: Popper.js is a library used to create popovers and tooltips, improving the user experience by providing additional information and context.

**Key Features & Functionalities**
- Accordion: The accordion component is used to display the three different appealing menu items on each website, allowing users to expand and collapse each item to view its details.
- Slider: A carousel slider is implemented using Bootstrap's carousel component to showcase different coffee items, providing an interactive and visually engaging experience for the users.
- Modal: The websites employ modal windows created with Bootstrap and powered by jQuery to display the contact us page and reservation form. Users can open these modals to interact with the forms without leaving the current page.
- Cart: The websites include a cart feature that allows users to add coffee items to their cart for ordering. This functionality is implemented using jQuery and customized CSS styles.
- Icons: Bootstrap's built-in icon library, along with custom icon sets, is utilized to add visual elements and enhance the user experience.
- Grid: The grid system provided by Bootstrap is used to create a responsive layout for the websites, ensuring optimal display on various screen sizes.
- Form: Bootstrap's form components, along with custom JavaScript and jQuery, are used to create user-friendly and interactive forms for the contact us page and reservation form.
- Checkbox and Input: Various form elements, such as checkboxes and input fields, are utilized to gather user input effectively and are enhanced with jQuery functionalities.
- Carousel, Tab, Collapse, Dropdown, Modal, Alert, List Group, Navbar: These Bootstrap components, along with custom JavaScript and jQuery, are leveraged to create dynamic and interactive sections of the websites, providing an intuitive user interface.
- Canvas Overlay: The websites incorporate a canvas overlay effect using custom JavaScript and jQuery, adding a visually appealing layer to the websites.
- Scroll: jQuery-based scrolling effects are implemented to enhance the user experience while navigating the websites.
- Container, Slide, Media, Card: Additional Bootstrap components such as containers, slides, media, and cards are utilized to structure and present the content effectively.
